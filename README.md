# Unity ML-Agents: Reinforcement Learning Project

> I was inspired by various youtube videos about training different agents and models using machine learning to do certain tasks.
> I witnessed a model learn how to play soccer or play red light green light, and I wanted to figure out how I can replicate that process myself to learn more about machine learning.
> I discovered that unity has its own machine learning agents toolkit, so I decided to tackle this project with that, allowing me to first learn from their examples and later progress to making my own.

## Project Goals
- Understand how reinforcement learning (PPO) works in practice
- Get Unity ML-Agents examples running with GPU-accelerated training
- Design and train a custom agent from scratch
- Document every step for reproducibility

## Tech Stack
- **Unity** 2022.3 LTS — simulation environment
- **Python** 3.10 — training orchestration
- **PyTorch** (CUDA 12.4) — neural network backend
- **ML-Agents** 0.30+ — Unity ↔ Python bridge
- **TensorBoard** — training visualization
- **Hardware** — NVIDIA RTX 5070 (GPU-accelerated training)
